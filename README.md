# 📱 Android Lab Final Project Collection

This repository contains a collection of Android development lab tasks built using **Kotlin**, **Firebase**, and modern Android tools. Each task demonstrates core concepts required for building real-world mobile applications.

---

## 🚀 Project Overview

This project includes the following lab tasks:

- 🔧 Task 11: User Settings Manager (SharedPreferences)
- 🔐 Task 12: Authentication System (Firebase Auth)
- 📚 Task 13: Course Manager (Firebase Realtime DB CRUD)
- 📰 Task 14: News App (Retrofit API Integration)

---

## 🧩 Task 11 — User Settings Manager

### 📌 Features

- Save user preferences using **SharedPreferences**
- Theme selection (Light / Dark / System)
- Notification toggle
- Language selection (English, Bangla, Arabic, French)
- Adjustable font size using SeekBar
- Profile setup with student details
- View saved settings with timestamp
- Reset settings to default

### 🛠 Technologies

- Kotlin
- SharedPreferences API
- ConstraintLayout, ScrollView

---

## 🔐 Task 12 — Student Authentication System

### 📌 Features

- User Registration (Email & Password)
- Login / Logout functionality
- Forgot Password via Email
- Splash screen with auth state check
- Secure session handling
- Change password & delete account
- Dashboard with user info

### 🛠 Technologies

- Firebase Authentication
- Kotlin
- Material UI Components

---

## 📚 Task 13 — Student Course Manager (CRUD)

### 📌 Features

- Add, View, Update, Delete courses
- Real-time data sync using Firebase
- RecyclerView with custom adapter
- Search courses by name/code
- Course detail view
- Confirmation dialogs for delete

### 🛠 Technologies

- Firebase Realtime Database
- Kotlin
- RecyclerView

---

## 📰 Task 14 — University News App

### 📌 Features

- Fetch posts from REST API
- View post details with comments
- User profile exploration
- Search posts by title
- Pull-to-refresh functionality
- Error & loading state handling

### 🌐 API Used

- https://jsonplaceholder.typicode.com/

### 🛠 Technologies

- Retrofit2
- Kotlin Coroutines
- Glide (Image Loading)
- MVVM Architecture
